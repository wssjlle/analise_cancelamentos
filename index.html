💼 Sobre o Projeto

Este projeto utiliza uma base de dados realista para entender os principais motivos pelos quais clientes cancelam serviços ou contratos. Através de análises exploratórias, gráficos e estatísticas, conseguimos obter insights valiosos que podem ajudar empresas a reter mais clientes.

Este projeto foi desenvolvido em Python, utilizando bibliotecas como Pandas, Seaborn e Matplotlib para realizar uma análise exploratória dos dados de cancelamentos de clientes.

🔍 Principais Insights

  • Clientes com alto número de reclamações tendem a cancelar mais.
  • Frequência de uso dos serviços está diretamente ligada à retenção.
  • Planos mais baratos têm maior índice de cancelamento.
